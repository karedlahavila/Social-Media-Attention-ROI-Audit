
# 📊 Social Media Attention ROI & User Segmentation Audit
Data-Driven Executive Strategy for Resource Allocation

## 🎯 Project Overview
In a landscape of vanity metrics, this project identifies the Economic Value of Time across major social platforms. Using a dataset of 1,000 users, we performed a behavioral audit to distinguish between high-value engagement and resource-intensive "vampire" behavior.

## 🧠 Technical Methodology
* Clustering:** Applied K-Means clustering to segment users based on `Daily_Minutes_Spent` vs. `Action_Index`.
* Metric Engineering:** Developed the `Attention_ROI` formula to calculate the conversion of time into platform actions.
* **Visualization:** Built an Executive "Dark Mode" Dashboard in Tableau for real-time strategic monitoring.

## Key Insights (The "So What?")
* The Global Baseline: The average user provides a 2.23 ROI.
* High-Efficiency Catalysts (25.4%):** These users are the profit engine, generating 5.05x return on every minute spent.
* Passive Vampire Drain (26.1%): A significant segment consuming high platform resources with a sub-1.0 ROI (0.44 avg).
* Strategic Pivot:** Data suggests a 10% budget shift toward Instagram and TikTok to optimize for high-efficiency personas.
